# WebOS

A Linux-style desktop environment that runs in the browser — and ships as a
native desktop app via Electron. Boot it up and you get a draggable window
manager, a taskbar and start menu, an installable app catalog, a working
terminal, and a shared virtual file system that persists across reloads.

Built with React 19 and packaged for Windows, macOS, and Linux with
electron-builder.

## Features

- Window manager — open, drag, resize, minimize, maximize, and focus multiple windows
- Desktop shell — desktop icons, a start menu, and a taskbar
- Installable app catalog and a working terminal
- Shared virtual file system that persists across reloads
