# Portfolio Website

A personal portfolio website built with React, showcasing projects, skills, and
experience. Features a light/dark/system theme, animated page transitions, and
MDX-powered project pages.

## Features

- Home, About, Projects, and Contact pages with a responsive layout
- Light / Dark / System theme switcher
- MDX-driven projects — each project is a single `.mdx` file that powers both the
  preview card and the full detail page
- Search, pagination, and tech-tag filtering on the projects page
- Animated transitions with a `prefers-reduced-motion` fallback
